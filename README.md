# Programação de Sistemas Computacionais com Python 🐍🖥️

Este repositório contém o desenvolvimento de ferramentas, scripts e módulos voltados para a **Programação de Sistemas Computacionais**. O objetivo principal é explorar a interação do Python com recursos de baixo nível do sistema operacional, gerenciamento de processos, manipulação de arquivos binários, concorrência e comunicação em rede.

---

## 🚀 Recursos do Projeto

* **Gerenciamento de Processos e Threads:** Uso de `multiprocessing` e `threading` para execução paralela e concorrente.
* **Comunicação em Rede:** Implementação de sockets (TCP/UDP) para comunicação entre processos (IPC).
* **Manipulação de Arquivos e I/O:** Leitura/escrita eficiente de buffers e manipulação de arquivos binários com o módulo `struct`.
* **Interface com o S.O.:** Chamadas de sistema e monitoramento de hardware via `os`, `sys` e `psutil`.

---

## 🛠️ Pré-requisitos e Dependências

Antes de começar, certifique-se de ter o **Python 3.10+** instalado em sua máquina.
